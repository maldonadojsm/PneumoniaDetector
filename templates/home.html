Description

Hello there! This web application intends to accurately classify bacterial or viral pneumonia using Deep-Learning Image Classification Methods. My model has been trained using 5800+ Chest X-Ray images provided by the Mendeley Organziation (https://data.mendeley.com/datasets/rscbjbr9sj/2). Models have been constructed using Tensorflow, Scikitlearn and Keras.

Usage

Users are prompted in the "Upload Images" tab to upload an X-Ray Lung Image in PNG or JPEG format. After submitting, the application will enter the image into the model; classifying the x-ray lung image as healthy or showing visual symptoms of either viral or bacterial pneumonia.
